# PWA Icons Needed

Please generate the following icon files and place them in the `public/icons/` directory:

1. **icon-192x192.png** (192x192 pixels) - Minimum size for Android
2. **icon-512x512.png** (512x512 pixels) - Recommended size for Android
3. **apple-touch-icon.png** (180x180 pixels) - For iOS home screen

## Icon Design Guidelines

- Use the SenTracker logo/wallet icon with emerald (#10b981) accent
- Ensure icons have transparent or white background
- Make icons "maskable" friendly (safe zone for Android adaptive icons)

## Quick Icon Generation Tools

- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

Place generated icons in: `public/icons/`
